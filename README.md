## Getting Started 🎲

> [!NOTE]
> Set up the environment variables file with the following variables.

```bash
# .env
GITHUB_TOKEN=

# Supabase
SUPABASE_URL=
SUPABASE_ANON_KEY=
```

> [!NOTE]
> Get `GITHUB_TOKEN` from [GitHub Personal Access Tokens](https://github.com/settings/tokens) for contribution stats.

### Run locally

```bash
pnpm install
pnpm dev
```

Open [http://localhost:1234](http://localhost:1234) with your browser.
